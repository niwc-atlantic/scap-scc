SCAP Compliance Checker (SCC) 5.12 Readme
09/05/2025

SCC_5.12_ReleaseNotes.txt
- Contains list of high level features added and bugs fixed in this release

SCC_5.12_Checksums.txt
- Contains SHA256 checksums of all zips, installers and binaries.

SCC_5.12_UNIX_Remote_Scanning_Plugin.scc
- Plugin which can be installed into the SCC GUI to perform SSH based remote reviews of Linux, Solaris and MacOS.

macOS:
- Contains release notes, user manual, and a tar.gz file for manual installation.  (package installer is planned for all future releases)
This build is designed to work on macOS 13, 14, 15 on Apple "M" processors.
